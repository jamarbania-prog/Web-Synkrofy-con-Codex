# Web Synkrofy

Base del proyecto web de Synkrofy según `docs/SYNKROFY-WEB-MASTER.md`.

## Stack

- Next.js 16 con App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Prisma con PostgreSQL

## Puesta en marcha

1. Instala dependencias:

```bash
npm install
```

2. Crea tu archivo `.env` a partir de `.env.example`.

3. Arranca el entorno de desarrollo:

```bash
npm run dev
```
